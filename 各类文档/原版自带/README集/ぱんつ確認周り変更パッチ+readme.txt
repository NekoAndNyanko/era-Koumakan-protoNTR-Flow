era紅魔館protoNTR用
	「ぱんつ確認周り変更パッチ+」

パッチの簡単な説明
	一定のコマンドの結果、奴隷が貞操帯を付けていることをあなたが確認した場合はぱんつ確認フラグが立つようになります。

導入手順
	「era紅魔館protoNTR0023」に「era紅魔館protoNTR0023用パッチまとめ」の「15_11_12」をあてた後に
	ERBフォルダを入れてください

アップロード日
	2015/11/13

修正・改造など
	修正・改造・バランス調整はご自由に


〜パッチ内容〜
	あなたが奴隷に対してCOM314「アナル愛撫」COM315「クリ愛撫」COM350「押し倒す」を選択した際
	指が貞操帯に阻まれる、奴隷に貞操帯を見せられるといった「あなたが直接貞操帯を確認した場合」に限り
	CFLAG:ぱんつ確認が1となります。
