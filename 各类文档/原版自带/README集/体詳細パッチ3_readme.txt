		下付き 性器が通常より少し後ろ(お尻の方)にある
	Ｖオプション（Ｖ特徴）外見じゃないような気もするけど
		数の子天井 Ｖ性交時の挿入側の射精ゲージ増加強化
		膣を自在に収縮できる 恋慕または淫乱で挿入側の射精ゲージ増加強化 意味合い的には淫壷に似てるような気もする
		
	Ｐ大きさ
		馬並み
		大きい
		普通
		小さい
		短小
		
	Ｐオプション（Ｐ特徴）
		真性包茎
		仮性包茎
		
		触手のように自在に動く
		全体的にイボが付いている
		ドリルチンポ

	Ｖオプションのみ行為に少し影響します（訪問者や客に影響するかはわかりません）他はフレーバー

	ふたなりになった場合は設定で指定したものがそのまま使われます
	
	子供の性器は女なら母親の、男なら父親のものが遺伝するわけではありません
	両方の親の設定しているものからランダムで遺伝します。
	つまり男だからといってＶ位置の設定をしていなければ
	母親が上付きでも子供(女)が必ず上付きになるわけではありません。
	

訪問者に外見の要素を追加できる(パッチ２)
	SHOP画面の設定→訪問者のステータス設定より訪問者の外見を設定できる
	 先ず外見要素の追加をonにすると選べる訪問者外見設定を選択すると訪問者の外見設定画面へ移動する
	但し髪型は設定した時に直ぐ指定の髪型になる
	（１日経過による髪が伸びる事は無い）
	もちろん子供への外見要素の遺伝も行われる
	 外見要素offだと遺伝情報はランダムとなる
	他のキャラとは違い何時でも変更可能

外見要素を追加
	設定は一度だけ変更可能
	一応それっぽい外見を設定しているのでそのままでも使えるかも
	
	髪オプションは髪が短い、又は風呂場で裸になったり、寝間着に着替えていると表示されません
	
	
髪が伸びる
	一日に一度伸びる判定がある
	いつ切るかはキャラ毎に違う
	
	
子供への外見要素の遺伝
	基本両親から半分ずつ貰う（時折変動する）
	髪の長さなど一部は後天的に決まる
	キャラとして設定されていないキャラの子はランダムで決まる
	
	
	オッドアイでも両目の色が同じなんて事もあるけど仕様です
