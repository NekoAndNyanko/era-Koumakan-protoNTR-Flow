-era紅魔館protoNTR用 訪問者が午前0時を越えると同じ行動を繰り返すマシーンになるのを防ぐパッチ-

簡単なパッチの説明
	長いパッチ名の通りです。
	163スレ >>238のバグ対応です。

導入手順
	【era紅魔館protoNTR Ver.0006】に【era紅魔館protoNTR修正パッチ130917】を適用した後に
	このパッチを上書きしてください
	
	
主な追加処理
	●访问者当前行为中止时刻の判定を、時刻から、日時に変更
	●と、時刻周りを読みやすく。
	●要睡眠時刻、睡眠可時刻をDIM.ERHの#DEFINEに移動。

作者様への要望
	●時刻を扱う場合は、拙作のDATETIME()関数をよろしく！
	　終了時刻判定をする際に、TIMEを使うと、0時を越える終了時刻を判定できません。
	●定数は#DEFINEに書いてほしいなぁと。特に複数ERBに渡るものは。


このパッチで使用している追加フラグ
	なし。
	けど、CFLAG:332の名前を変更。
