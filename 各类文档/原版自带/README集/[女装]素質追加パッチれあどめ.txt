〜パッチ内容〜
	あなたが素質[オトコ]を持っていて(正確には女かふたなりでない場合)、服装から女だと誤解できる見た目をしていれば
	着替えた直後に素質[女装]が付与されるようになります。
	あくまで傍から見たら女と誤解できるかどうかを基準としているためコスプレをしているだけでは付与されません。
	また、全身タイツなどは体つきがはっきり出て女と誤解できないだろうと考えたため付与されません。
	
	少し弄れば他の紅魔館住人に対しても付与できますが、住人をオトコにした遊び方が想定されていないため今のところあなた以外には付与されません。
	この素質によりあなたが女装をしている時用の口上等(男なのに女の子みたいな恰好をして〜みたいな)を作るのが楽になると思いますが、
	現状はただのフレーバーとしての意味しかありません。
	
	
	[女装]が付与される条件は以下の通りです。
	前提条件：あなたが男であること
	1.スカートを履いていて上半身が裸でないこと(体つきを示す素質などは今のところないが、上半身が裸だとさすがに男だとわかるだろうという考えから)
	2.ワンピースを着ていて下半身上着を履いていないこと(ワンピースのチャイナ服のみズボンを合わせて履くことができ、中華服+ズボンはユニセックスなものであるため)
	
	[女装]が付与される服は以下の通りです。
	・ミニスカート,スカート,ロングスカート,タイトスカート,裳,ブルマスカート・・・・・・上半身が裸でなければ[女装]が付与されます
	・ワンピース,修道服,魔女のドレス・・・・・・着ていれば[女装]が付与されます
	・チャイナ服・・・・・・ズボンを履いていない場合のみ[女装]が付与されます
