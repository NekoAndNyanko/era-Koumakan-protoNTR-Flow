era紅魔館protoNTR用部屋のにおいパッチ

はじめに
	私はふと思ったのです。ゴミの山…この部屋って超臭くないのか、と。
	しかしにおい(臭いと匂い、どちらの意味でも)の情報を表示させるならアレした痕跡が残ってもいいんじゃね？と。

このパッチをあてると？
	部屋のにおいを表示するようにできます。オプションでの ON/OFF 切り替えも可能。
	においを嗅ぐコマンドも追加され、自分や各住人のにおいを嗅ぐ事ができます。
	お気に入りのキャラのにおいを嗅いだら別の住人のにおいが……？

使用フラグと仕様
	FLAG
		500,ON/OFF
		500+部屋番号,各部屋のにおいの種類と強さ
		550,前回のにおいチェック時間
		550+部屋番号,各部屋のにおいの種類と強さ
		600,訪問者の他人のにおい
	CFLAG
		800,各キャラの体臭の強さ
		801,各キャラ自身のにおい
		802-704,各キャラについた他人のにおい
		805,翌日の朝にリセットされる中出しされた精液のにおいの強さ(最も新鮮なもののみ)
		805+キャラ番号,各キャラのにおいを嗅いだ回数
	またしてもえらい量のフラグを使ってしまってすみません…。
	部屋には二人分のにおいを、各キャラには自分と三人分の他人のにおいを残せるようにしてあります。
	一つにまとめることもできるのですが…
	例えばAの精液とBの愛液をひとつのフラグにまとめるとどっちの精液でどっちの愛液かという情報を残せないので…。
	DIM.ERH の方で変更可能。

改変者向けメモ
	0987654321
		・1〜5桁目は部屋のにおいの種類
			0:無臭
			1(bit1):ゴミ
			2(bit2):汗
			4(bit3):体臭(臭いって事じゃないから！香水とかシャンプーとかそういうのの残り香だから！)
			8(bit4):愛液
			16(bit5):精液
		・6〜7桁目はにおいの強さ
		・8〜桁目はキャラ番号
		例：FLAG:508が21500006の場合、小悪魔の部屋(部屋番号8)で小悪魔(キャラ番号2)の強さ15の体臭と汗のにおいがする
		ビットなので複数種類のにおいを残すことも可能だがそれぞれの強さは残らない。まぁ…混ざった強いにおいということで。
