era紅魔館protoNTR用アリス追加パッチ+


パッチの内容

アリスを追加（行動時間6時〜22時）、アリスの家を追加
それにともなったその他キャラの行動などの変更
魔理沙、霊夢、ルーミアの設定漏れと思われる場所の追加
あなたの体詳細初期設定を調整（髪と目の色を黒に、肌の色を普通に、髪の長さを調整）
チルノの家、大妖精の家、魔理沙の家の風呂場設定を削除


チルノの家、大妖精の家、魔理沙の家の風呂場設定の削除理由

テストプレイしたさい、チルノの家、大妖精の家、魔理沙の家、アリスの家にお風呂が設定されている状態でテストしたのですが
キャラが以上の場所でお風呂に入っていて全裸であることが多く、また貞操帯鍵購入時にお風呂の判定がある場所だと
貞操帯の解除ができないため不便、またそれぞれのキャラが大浴場にいくよう設定されていたため
チルノの家、大妖精の家、魔理沙の家の風呂場設定を削除しました


導入手順

era紅魔館protoNTR0025にera紅魔館protoNTR0025用パッチまとめ16_07_05を適用したうえで
このパッチを導入ください


注意書き、お願い

改造、改変、流用などご自由にどうぞ

今回、ファイル内テキストの検索ツールを使ってバリアントの他のキャラの設定などをコピペ、改変して
アリスを追加できるパッチを製作しました
ほとんどのところがどのような処理がされているかわかりやすかったのでうまく製作することができましたが
このパッチの作者は構文がわからないため
作者の能力を越えて複雑だったTOILET_COUNTER_ACTABLE.ERB、TOILET_COUNTER_MESSAGE.ERB内の処理は
できる範囲で他のキャラの処理があるところにアリスの処理を追加しました
RESTROOM.ERBは魔理沙、ルーミア、霊夢も設定されてないこと、構文もよく分からないこと
そしてトイレの設定とかどうしたらいいのか分からなかったため、設定していません
また、CLOTHES_SYSTEM.ERBとCOMF430.ERBを使用したキャラのセットに着せ替えができるようになっていますが
子悪魔以降、追加されていないみたいなので追加していません
トイレの設定、着せ替えのセット、必要な方いましたらどうぞ設定、追加なさってください

あとがき

この度、era紅魔館protoNTRにアリスを追加しようと思い立ち、はじめてこのようなパッチを製作したのですが
アリスが魔法の森に住んでいること、すでに魔理沙を実装した方、ルーミア・霊夢を追加した方がおられたこと
そして構文が分からない中でどんな処理がされているか分かりやすく、コピペだけでやっていけたこと
以上の幸運に恵まれて容易にこのパッチを製作できました
このパッチがすこしでもお役にたてれば幸いです
素晴らしいバリアントの作者様、またこれにかかわるすべての方々に感謝いたします
