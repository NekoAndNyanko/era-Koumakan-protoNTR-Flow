-era紅魔館protoNTR用多生児出産と(ryパッチ V1.4Fix-


簡単なパッチの説明
	親や子供、兄弟の関係を能力表示画面に表示したり
	ついでに相性も表示したり
	双子とか三つ子とかが産まれるようになるパッチ
	

導入手順
	"era紅魔館protoNTR Ver.0.020"に"era紅魔館protoNTR0020用パッチまとめ15_05_02"を適用した後に
	このパッチを入れてください
	
	
主な追加処理
▲v1.4Fix
	●能力表示の兄弟関係にて多生児の表記が固定してしまう不具合の修正
	
▲v1.4
	●異父二(多)卵性双生児の処理を追加
	　双子以上を出産する際、妊娠した時に父親になる可能性があるキャラが複数いた場合
	　父親の違う子供が生まれる事が起こる様になった
	
▲v1.3
	●子供が産まれた際の相性設定を追加
	　両親との相性が300になる
	　両親の他のキャラとの相性は以下の処理を行ったうえで受け継がれる
	　
	　1,受け継ぐ相性が両親共に同じだった場合 (父親の美鈴との相性が150 母親の美鈴との相性が150のような場合)
	　  受け継ぐ相性が100超なら ((受け継ぐ相性-100)/2)+100
	　    (親が150なら子供は125となる)
	　  受け継ぐ相性が100未満なら((100-受け継ぐ相性)/2)+受け継ぐ相性
	　    (親が80なら子供は90となる)
	　  が子供に受け継がれる相性の値となる
	　
	　2,受け継ぐ相性が両親によって違う場合 (父親の美鈴との相性が150 母親の美鈴との相性が200のような場合)
	　  双方の相性の平均値が子供に受け継がれる相性の値となる
	　
	　片親が登録外のキャラ(訪問者や客)の場合、1,の処理を行う

・複数人の父親の子を産む（※V1.4適用前に妊娠したキャラは対象外）
　１,妊娠した時に複数人の精子を子宮内に取得している
　
　２,双子以上を産む場合、一人ずつ子供の生成処理を行い、その際に一つ前に生成した子供と同じ受精卵であるかを設定する
　　１,の条件を満たしていると、同じ受精卵でない子供が出来る度に
　　次以降の子供の父親を変更する処理が入る
　

１,の条件を満たしている且つ双子以上を産む状態で里子に出した場合、
生まれる子供は全て第一子の父親として処理される

・父親を変更する処理
　父親の変更確率は母親となるキャラが妊娠した時に子宮内に取得していたキャラごとの精子量が関わる
　精子量が多いキャラほどその子供の父親になりやすくなる

・複数の父親がいる場合の父親の命名権
　プレイヤーキャラが父親の場合は、産まれてくる一人目の子の父親がプレイヤーキャラである場合
　以降の子供も含め全ての子供をプレイヤーが命名することが出来ます

　プレイヤーキャラが母親の場合は、産まれてくる一人目の子の父親となるキャラに
　以降の子供も含め全てそのキャラに命名権が渡ります

・ステータス表示の兄弟関係の項目で父親の違う多生児兄弟の表示を変更
　"双子"などの後ろに"/異父"がつく


尚、地の文関係は一切触っていないので、場合によっては内容がおかしくなるかもしれない
