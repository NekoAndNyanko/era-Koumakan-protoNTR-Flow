- 訪問者設定パッチ (era紅魔館protoNTR用)　勝手に改造版

お気に入りのキャラ、嫌いなキャラ、気になる場所、お気に入りの場所を
ランダムで選ぶ選択肢を追加。
訪問者にも選択の自由を！

元パッチを導入後に上書きしてください。

//使用フラグ番号 元パッチ使用分
FLAG:1800 訪問者 お気に入りのキャラ
FLAG:1801 訪問者 嫌いなキャラ
FLAG:1810 訪問者 気になる場所
FLAG:1811 訪問者 お気に入りの場所

//使用フラグ番号　改造追加分
FLAG:1820 訪問者 お気に入りのキャラのランダムフラグ
FLAG:1821 訪問者 嫌いなキャラのランダムフラグ
FLAG:1830 訪問者 気になる場所のランダムフラグ
FLAG:1831 訪問者 お気に入りの場所のランダムフラグ

仕様で悩んでる点
ランダムお気に入りと手動嫌いが被ったときの動き。
現状では元パッチの動きに便乗して手動の設定を優先、
ランダムは-無し-にするように。
どうせなら自動でお気に入りを選びなおした方が親切？
でも嫌い設定中にお気に入り判定を走らせるのはスパゲティ化が怖い。
