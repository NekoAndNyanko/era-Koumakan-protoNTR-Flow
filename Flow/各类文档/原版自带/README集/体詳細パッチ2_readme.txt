・不具合修正
1.髪や目の色を選ぶ時、栗色だけは色の部分まで入っているので栗色色になる
	文字を変更して修正
2.特徴やその他の空欄部分を1度選ぶと取り消し等が無いので消す事ができない(別の特徴で上書きは可能)
	特徴選択画面に 戻る の上に [0] リセット を追加


訪問者に外見の要素を追加できる(new)
	SHOP画面の設定→訪問者のステータス設定より訪問者の外見を設定できる
	 先ず外見要素の追加をonにすると選べる訪問者外見設定を選択すると訪問者の外見設定画面へ移動する
	但し髪型は設定した時に直ぐ指定の髪型になる
	（１日経過による髪が伸びる事は無い）
	もちろん子供への外見要素の遺伝も行われる
	 外見要素offだと遺伝情報はランダムとなる
	他のキャラとは違い何時でも変更可能







外見要素を追加
	設定は一度だけ変更可能
	一応それっぽい外見を設定しているのでそのままでも使えるかも
	
	髪オプションは髪が短い、又は風呂場で裸になったり、寝間着に着替えていると表示されません
	
	
髪が伸びる
	一日に一度伸びる判定がある
	いつ切るかはキャラ毎に違う
	
	
子供への外見要素の遺伝
	基本両親から半分ずつ貰う（時折変動する）
	髪の長さなど一部は後天的に決まる
	キャラとして設定されていないキャラの子はランダムで決まる
	
	
	オッドアイでも両目の色が同じなんて事もあるけど仕様です
